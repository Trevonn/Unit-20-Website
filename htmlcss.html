    HTML (Hyper Text Markup Language) is a markup language used to document the structure of web pages. HTML uses tags for this. For example all the content of the page is contained in the body tag. There is an example of this below

    <h1>This is a level one heading</h1>

    CSS
    CSS (Cascading Style Sheets) is a style sheet language. It's main purpose is controlling the layout and presentation of web pages it uses piece of code called a selector to refer to elements in the html doucment. Inside the selector is where the rules go. Rules are short and only take up one line each. There is an example of a rule and selector below

    p {

    font-weight: bold;

    }

    The selector above is called a HTML Selector because selects HTML elementsThis rule selects all <p></p> tags in the linked html document and changes the text to a bold font. There are also CSS Selectors which I will give an example of below

    .text-bold {

    font-weight: bold;

    }

    The benefit of this approach is that this code is applied to any HTML tag with a class attribute of text-bold instead of just all <p></p> tags giving you the flexibility to add the text-bold class to any html element.
