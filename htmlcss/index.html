    • HTML

    HTML (Hyper Text Markup Language) is a markup language used to document the structure of web pages. HTML uses tags for this. For example the title and subheadings of an article would use header tags. There are code examples of this below.

    Code Example

    <h1>This is a level one heading</h1>

    <h2>This is a level two heading</h2>

    <h3>This is a level three heading</h3>

    <h4>This is a level four heading</h4>

    <h5>This is a level five heading</h5>

    <h6>This is a level six heading</h6>

    HTML has tags for many other article features such as lists with <ul></ul> and <ol></ol>.

    CSS

    CSS (Cascading Style Sheets) is a style sheet language. It's purpose is controlling the layout and presentation of web pages it uses a piece of code round called a selector to refer to elements in the html doucment. Inside the selector is where the rules go. Rules are short and only take up one line each. There is code example of a selector with a rule inside below. CSS Stylesheets are interpreted from top to bottom this means some rules can be inherited

    Code Example

    p {

    font-weight: bold;

    }

    The selector above is called a HTML Selector because it selects HTML elements using their tag name. The rule selects all <p></p> tags in the html document and changes the text to a bold font. There are also CSS Selectors which I will give code example of below.

    Code Example

    .bold {

    font-weight: bold;

    }

    The benefit of this approach is that this code round is applied to any HTML tag with a class attribute of bold instead of just all <p></p> tags giving you the flexibility to add the bold class to any html element.
