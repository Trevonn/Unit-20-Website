    • CSS Stylesheets

    There a three different ways to add CSS stylesheets to a page. Each method has it's advantages and disadvantages. I will explain in detail each method below.

    Internal Stylesheets

    Internal Stylesheets are CSS styles embedded within html tags. For example the follwing text will be red. Red Text.

    Code Example

    <span>style="color: red;">Red Text</span>

    This method of applying CSS Styles is very inefficient because if you want to apply the CSS Styles to multiple instances of the same element the style must appear in the every instance of the element.

    Embedded Stylesheets

    Embedded Stylesheets are CSS styles that are placed in the header tag of the HTML document.

    Code Example

    <style>

    p {

    color: red;

    }

    </style>

    This method of adding CSS Styles to a page is only beneficial on one page websites. This is because when a web page is loaded it will request the website's resources from the server. If the website uses an external CSS stylesheet there will be two requests one for the HTML file and one for the CSS file. Whereas with an embedded stylesheet there will only be one request. This means the webpage will load faster because there are fewer requests.

    External Stylesheets

    External Stylesheets are external css files linked in the header of the HTML document.

    Code Example

    <link href="css/style.css" rel="stylesheet">

    @charset "utf-8";

    p {

    color: red;

    }

    This is an example of an externally linked stylesheet. Externally linked stylesheets are better for websites with multiple pages because each page is using the same stylesheet. This saves space and bandwidth because you don't have to copy the styles onto each webpage.
